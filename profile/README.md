## 🎯서비스 소개

운동은 하고 싶은데 종류도 너무 많고 기구도 너무 많아.. 뭐가 뭔지 하나도 모르겠어...🥹

하나씩 다 알아보기에는 피곤하고, 시간이 아까워.

이런 당신에게 필요한 것, 바로 **Fitmate** 입니다!

### 간단한 정보 입력만으로 내 몸에 맞는 운동을 찾아보세요! 🔍

### 내게 맞는 운동을 찾았다면 루틴을 만들어 관리해보세요! 🛞
